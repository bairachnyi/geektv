The GeekMagic SmallTV is a small cube with a 1.54" colour screen, an ESP inside, and a USB-C port. This development firmware currently has three modes you switch between in a built-in web UI, and it updates over WiFi. The web UI header shows a chip badge for the board it's running on, and an optional scheduled night mode dims or turns off the backlight on an NTP clock and timezone you pick, without taking the device offline.

:::caution[Factory features are not ported yet]
Clock themes, weather/forecast and the photo gallery from factory V9.0.51 are not
in this candidate. Keep testing in the emulator; do not flash it as a complete
factory replacement yet.
:::

## The three modes

<CardGrid>
  <Card title="Stock and crypto ticker" icon="seti:default">
    Price, change, percent change, and a sparkline for up to 8 rotating symbols. Data from Yahoo Finance with no backend, or your own webhook.
  </Card>
  <Card title="AI usage meter" icon="seti:crystal">
    Antigravity and Codex percentages with reset countdowns, fed by a trusted local bridge. Personal-plan values require a local adapter because the providers do not expose public personal-quota APIs.
  </Card>
  <Card title="GitHub GH//STAT" icon="seti:github">
    Animated Actions, deployments, pull-request checks and releases with live timers, timestamps, pagination and structured errors for private repositories.
  </Card>
</CardGrid>
